# Datastory — NFL Player Medical Profiles

> **This is a demo. It contains no confidential data/IP.**

## Overview

This repository contains synthetic medical profile data for 10 fictional NFL players. It is designed to showcase a data storytelling use case around player health, injury risk assessment, and medical conflict analysis in a professional sports context.

## Contents

| File / Folder | Description |
|---|---|
| `player_medical_profiles.json` | Structured JSON containing medical data for all 10 synthetic players |
| `Synthetic_data/` | Individual Markdown reports per player with detailed medical findings |
| `info.txt` | Summary of conflicts, red flags, and critical findings across all players |

## Key Themes

Each player profile documents synthetic medical imaging findings, injury history, and risk flags, including:

- **Structural injuries** (fractures, labral tears, osteochondral defects)
- **Neurological risks** (cervical stenosis, concussion history, stingers)
- **Cardiac conditions** (HOCM — life-threatening in Tyrell Biggs)
- **Degenerative conditions** (chondromalacia, osteoarthritis, disc desiccation)
- **Systemic health** (renal disease, hypertension, diabetes)
- **Kinetic chain disruptions** (lumbar → hamstring, shoulder → elbow, hip → groin)

## Disclaimer

All player names, IDs, medical records, and findings in this repository are **entirely fictional and synthetically generated**. Any resemblance to real persons is coincidental. This data is intended for demonstration and storytelling purposes only.
